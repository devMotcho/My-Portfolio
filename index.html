DataMotive

A Django-based Inventory Management System designed to simplify the tracking of products, stock, and transactions in a user-friendly interface. It features CRUD operations for products, categories, partners, and transactions, while ensuring real-time updates to stock levels and pricing. With a dashboard displaying KPIs via Chart.js, pagination, and custom alerts, it offers a streamlined user experience. The system leverages Django signals to manage product stock and user profiles. It requires user authentication, integrates PostgreSQL for data storage, and is containerized using Docker for easy setup. The frontend utilizes HTML, Bootstrap, CSS, and JavaScript.

Sales Data Analysis

A sales data analysis project built using Pandas Python, focusing on data manipulation and visualization. It processes sales orders, groups data by products, and computes various metrics such as product quantities and average prices. The project also includes functionalities like plotting bar charts of sales quantities versus product prices and exporting the results into Excel with embedded charts. Key libraries utilized are pandas for data analysis and matplotlib for visualizations, while Jupyter Notebook facilitates the interactive development of the solution. The final output includes a visual and data-driven representation of sales performance.
